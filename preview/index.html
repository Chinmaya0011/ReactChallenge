JavaScript Problems for React Projects

Day 1: Basics and Data Types

  • 1. Sum of an Array
    Write a function that takes an array of numbers and returns their sum. This is a basic exercise to work with arrays and loops.
  • 2. Reverse a String
    Write a function that reverses a given string. This helps practice string manipulation and loops.
  • 3. Find Maximum Element in Array
    Write a function that finds the maximum element in an array. It helps in understanding how to traverse arrays and compare elements.
  • 4. Check if a String is a Palindrome
    Write a function to check if a string is the same when reversed. It will help understand string manipulation and comparison techniques.
  • 5. Array Flattening
    Write a function that flattens a nested array to a single level. This is useful in data processing when dealing with nested structures.

Day 2: Object and Array Manipulation

  • 6. Deep Clone an Object
    Write a function to create a deep clone of an object. This helps with understanding the concept of shallow vs deep copies.
  • 7. Merge Two Arrays
    Write a function that merges two arrays without duplicates. This teaches array manipulation and working with unique data.
  • 8. Filter Odd Numbers
    Write a function that filters out odd numbers from an array. This introduces the use of filter methods on arrays.
  • 9. Sort an Array of Objects
    Write a function that sorts an array of objects by a property. This is a common operation in React apps when displaying lists of data.
  • 10. Map an Array to Another Format
    Write a function that converts an array of strings to an array of objects. It helps practice the `map()` function.

Day 3: Functions and Loops

  • 11. Debounce a Function
    Write a debounce function to limit the number of times a function is called in quick succession. This is a key concept for performance optimization in React apps.
  • 12. Throttle a Function
    Write a throttle function to limit the number of times a function can be called over a period. It's useful for handling high-frequency events like scrolling.
  • 13. Count Occurrences of a Character
    Write a function to count how many times a character appears in a string. This teaches the use of string functions and loops.
  • 14. Factorial of a Number
    Write a recursive function to find the factorial of a number. This helps practice recursion and understanding its benefits.
  • 15. Fibonacci Sequence
    Write a function to generate the first N Fibonacci numbers. This introduces working with sequences and recursion.

Day 4: DOM and Event Handling

  • 16. Add Event Listener to Multiple Elements
    Write a function that adds a click event listener to multiple DOM elements. This helps in batch handling of DOM events.
  • 17. Get Element Position in Viewport
    Write a function that returns the position of an element relative to the viewport. This can be useful in scroll-related tasks.
  • 18. Create a Dynamic HTML Table
    Write a function that creates a table dynamically based on input data. This is important for building interactive, data-driven React apps.
  • 19. Toggle a Class on Click
    Write a function that toggles a class on an element when clicked. This is useful for interactive UI elements like buttons or modals.
  • 20. Prevent Default Form Submission
    Write a function to prevent form submission and log the form data to the console. It's a common pattern for handling forms in React.

Day 5: Asynchronous JavaScript

  • 21. Promise Chain
    Write a function that chains multiple promises and logs the result. This is a key concept for handling asynchronous code in React.
  • 22. Fetch Data from an API
    Write a function that fetches data from an API and logs the result. It teaches working with APIs, which is crucial for React apps.
  • 23. Handle Errors in Asynchronous Code
    Write a function that fetches data and handles errors if the fetch fails. This is essential for writing robust React components.
  • 24. Async/Await with Fetch
    Rewrite the fetch function using async/await syntax. This helps improve readability and error handling in asynchronous code.
  • 25. Delay Execution Using setTimeout
    Write a function that delays the execution of a task using setTimeout. This introduces working with time delays in JavaScript.
