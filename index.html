JavaScript Problems for React Projects

Day 1: Basics and Data Types

  • 1. Sum of an Array
    Write a function that takes an array of numbers and returns their sum. This is a basic exercise to work with arrays and loops.
  • 2. Reverse a String
    Write a function that reverses a given string. This helps practice string manipulation and loops.
  • 3. Find Maximum Element in Array
    Write a function that finds the maximum element in an array. It helps in understanding how to traverse arrays and compare elements.
  • 4. Check if a String is a Palindrome
    Write a function to check if a string is the same when reversed. It will help understand string manipulation and comparison techniques.
  • 5. Array Flattening
    Write a function that flattens a nested array to a single level. This is useful in data processing when dealing with nested structures.

Day 2: Object and Array Manipulation

  • 6. Deep Clone an Object
    Write a function to create a deep clone of an object. This helps with understanding the concept of shallow vs deep copies.
  • 7. Merge Two Arrays
    Write a function that merges two arrays without duplicates. This teaches array manipulation and working with unique data.
  • 8. Filter Odd Numbers
    Write a function that filters out odd numbers from an array. This introduces the use of filter methods on arrays.
  • 9. Sort an Array of Objects
    Write a function that sorts an array of objects by a property. This is a common operation in React apps when displaying lists of data.
  • 10. Map an Array to Another Format
    Write a function that converts an array of strings to an array of objects. It helps practice the `map()` function.

Day 3: Functions and Loops

  • 11. Debounce a Function
    Write a debounce function to limit the number of times a function is called in quick succession. This is a key concept for performance optimization in React apps.
  • 12. Throttle a Function
    Write a throttle function to limit the number of times a function can be called over a period. It's useful for handling high-frequency events like scrolling.
  • 13. Count Occurrences of a Character
    Write a function to count how many times a character appears in a string. This teaches the use of string functions and loops.
  • 14. Factorial of a Number
    Write a recursive function to find the factorial of a number. This helps practice recursion and understanding its benefits.
  • 15. Fibonacci Sequence
    Write a function to generate the first N Fibonacci numbers. This introduces working with sequences and recursion.

Day 4: DOM and Event Handling

  • 16. Add Event Listener to Multiple Elements
    Write a function that adds a click event listener to multiple DOM elements. This helps in batch handling of DOM events.
  • 17. Get Element Position in Viewport
    Write a function that returns the position of an element relative to the viewport. This can be useful in scroll-related tasks.
  • 18. Create a Dynamic HTML Table
    Write a function that creates a table dynamically based on input data. This is important for building interactive, data-driven React apps.
  • 19. Toggle a Class on Click
    Write a function that toggles a class on an element when clicked. This is useful for interactive UI elements like buttons or modals.
  • 20. Prevent Default Form Submission
    Write a function to prevent form submission and log the form data to the console. It's a common pattern for handling forms in React.

Day 5: Asynchronous JavaScript

  • 21. Promise Chain
    Write a function that chains multiple promises and logs the result. This is a key concept for handling asynchronous code in React.
  • 22. Fetch Data from an API
    Write a function that fetches data from an API and logs the result. It teaches working with APIs, which is crucial for React apps.
  • 23. Handle Errors in Asynchronous Code
    Write a function that fetches data and handles errors if the fetch fails. This is essential for writing robust React components.
  • 24. Async/Await with Fetch
    Rewrite the fetch function using async/await syntax. This helps improve readability and error handling in asynchronous code.
  • 25. Delay Execution Using setTimeout
    Write a function that delays the execution of a task using setTimeout. This introduces working with time delays in JavaScript.

Day 6: React Essentials

  • 26. Conditional Rendering
    Write a React component that conditionally renders a message based on a prop. Example: If isLoggedIn is true, render "Welcome", otherwise render "Please log in."
  • 27. Handling State in React
    Write a React component that uses the useState hook to toggle visibility. Example: Toggle a modal or a dropdown menu.
  • 28. Passing Props to a Child Component
    Write a parent and child component where the parent passes a value to the child. Example: The parent passes a name to the child to display it.
  • 29. Event Handling in React
    Write a React component that handles a button click event. Example: Display an alert message when a button is clicked.
  • 30. Form Handling in React
    Write a React component that handles form submission and displays the entered data. Example: Use useState to capture user input and display it upon form submission.

Day 7: React Component Lifecycle

  • 31. Use useEffect for Data Fetching
    Write a component that fetches data in useEffect and displays it. Example: Fetch user data when the component mounts and display it.
  • 32. Cleanup in useEffect
    Write a React component that cleans up after itself when it unmounts. Example: Use useEffect to remove an event listener when the component is removed.
  • 33. ComponentDidMount Equivalent in React
    Write a component using useEffect to mimic componentDidMount behavior. Example: Fetch data when the component mounts.
  • 34. Use useEffect to Watch Props Changes
    Write a React component that updates state based on prop changes. Example: Update a counter when a prop count changes.
  • 35. Use useRef to Access DOM Elements
    Write a React component that uses useRef to focus an input field on mount. Example: Focus on an input element when the component renders.

Day 8: Advanced JavaScript and React

  • 36. Custom Hook for Local Storage
    Write a custom React hook to manage state in local storage. Example: Use local storage to persist a theme preference.
  • 37. Render Props Pattern
    Write a component using the render props pattern to pass data to children. Example: A component that provides a state to its children through a render function.
  • 38. React Context API for State Management
    Write a basic implementation of Context API to manage global state. Example: Provide a theme value through Context and consume it in child components.
  • 39. Dynamic Component Rendering
    Write a React component that dynamically renders different components based on props. Example: Render a Card or Button component based on the type prop.
  • 40. Memoization in React
    Write a component using React.memo to avoid unnecessary re-renders. Example: Use React.memo for a list component that doesn't need re-rendering unless the props change.

Day 9: Advanced JavaScript Concepts

  • 41. Currying
    Write a function that uses currying to add multiple numbers. Example: add(1)(2)(3) should return 6.
  • 42. Closure Example
    Write a function demonstrating closures with a private counter. Example: counter() should return a function that increments a count.
  • 43. Bind Context in JavaScript
    Write a function that binds context using bind(). Example: Use bind() to bind a method to a specific object.
  • 44. Destructuring Assignment
    Write a function that destructures an object and array and returns a new object. Example: Destructure { name: "John", age: 30 } into variables.
  • 45. IIFE (Immediately Invoked Function Expression)
    Write an IIFE that logs a message to the console. Example: (function() { console.log("Hello") })() should log "Hello".

Day 10: Error Handling and Optimization

  • 46. Handle Multiple Errors Gracefully
    Write a function that handles multiple types of errors and provides feedback to the user. Example: Use try-catch blocks to handle different errors and display user-friendly messages.
  • 47. Memoization Function for Performance
    Write a memoized version of a function that calculates Fibonacci numbers. Example: fib(5) should return 5 and cache the result.
  • 48. Prevent Memory Leaks in React
    Write a React component that prevents memory leaks when making asynchronous calls. Example: Use cleanup functions inside useEffect to cancel ongoing API requests when the component unmounts.
  • 49. Lazy Loading in React
    Write a React component that loads a component lazily using React.lazy. Example: Dynamically load a component only when needed.
  • 50. Code Splitting with React
    Write a React app that implements code splitting using React.Suspense. Example: Use React.Suspense to load components asynchronously.
